# Bookshelf Web App

## Description

Bookshelf is a simple app for storing and searching for books.

## Dependencies

- Node.js (v20.13.1)
- npm (v10.5.2)
- Bookshelf API (v1.0.0)

## Installation

```bash
npm install
```

## Build

### Build Web

```bash
npm run build
```

## Usage

This will start the Web app.

### Start Web

```bash
npm start
```

## Development

This will start the Web app in development mode.

### Start Web

```bash
npm run dev
```
